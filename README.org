** how to use: 
1. Download and modify env file
#+BEGIN_SRC sh
cd ~ && git clone https://github.com/shdennlin/dotfiles.git ~/.dotfiles/
cd ~/.dotfiles
cp .env.example .env
vim .env
#+END_SRC

2. Install
#+begin_src sh
./install.sh
#+end_src

** env file config
#+begin_src sh
computer_type=m # master or slave
git=y # yse or no
alias_file=y # yse or no
zsh=y # yse or no
